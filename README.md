# telegram_bot
telebot with api(btc)

## installation:
  - 1.pip install -r requirements.txt
  - 2.create bot in telegram by botfather
  - 3.create file auth_data.py and copy your token into a token variable ( token = '5gfdgd4tgdfgdfgdf')
  - 4.launch
